      About This Page

      This page is part of the SecureNet Phishing Detection System, which helps users identify whether a website URL is safe or potentially malicious. Phishing websites are designed to trick users into revealing sensitive information such as passwords, banking credentials, or personal data by imitating legitimate websites.

      When a user enters a URL, the system analyzes several security factors including whether the website uses a secure HTTPS protocol, the length of the URL, suspicious keywords, and the number of subdomains. Based on these checks, the system calculates a risk score that indicates the likelihood of the website being unsafe.

      The results are displayed using risk indicators, detection reasons, website details such as domain, IP address, and country information, along with a visual chart showing security metrics like accuracy, precision, recall, and security score. This interface helps users quickly understand whether the website can be trusted.

      The goal of this system is to improve cybersecurity awareness and help users avoid phishing attacks by allowing them to analyze suspicious links before visiting them.
